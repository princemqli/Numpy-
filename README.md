# Python AI/ML/Data Science Self Exercises

## 📌 Project Overview

This repository contains a collection of **Python, AI, Machine Learning, and Data Science practice exercises** designed to strengthen fundamental concepts required for data analysis and scientific computing.

The notebook focuses on understanding important tools and techniques used in the Data Science workflow, including **NumPy operations, Jupyter Notebook environment, Google Colab, and array manipulation techniques**.

This project is created as part of hands-on learning and skill development in Python for Data Science.

---

## 🎯 Objectives

* Understand the fundamentals of NumPy in Python.
* Learn how Jupyter Notebook supports data analysis workflows.
* Explore Google Colab for cloud-based Python programming.
* Practice creating and manipulating NumPy arrays.
* Understand indexing and slicing techniques.
* Develop a strong foundation for AI/ML and Data Science projects.

## 🚀 How to Run the Project

### Using Jupyter Notebook

1. Install Python on your system.
2. Install required libraries:

```bash
pip install numpy jupyter
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Run the notebook cells step-by-step.

---

### Using Google Colab

1. Open Google Colab.
2. Upload the `.ipynb` file.
3. Execute cells to run the exercises.
